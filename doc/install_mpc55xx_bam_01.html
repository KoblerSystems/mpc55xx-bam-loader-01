2. Install Python

Follow the instructions on https://www.python.org to install Python. Select the latest stable version of Pyhton 3.

Install support for serial port

python -m pip install pyserial

You can find more information about pyserial on http://pythonhosted.org/pyserial.

3. Test with Phytec phyCORE-MPC5554

You need to set the Jumper JP1 to the memory, which has been erased. When BAM cannot start an application in the selected memory, then it listens on the eSCI_A and on the CAN port.

Connect the serial cable to the lower serial port.
